# CSE160 Assignment 5 — *Up* Adventure World

## Author

Hegen Chang

A Three.js 3D scene inspired by Pixar’s [*Up*]: Carl’s balloon-lifted house, Paradise Falls, a Spirit of Adventure–style blimp, animated wildlife, and a large explorable environment.

## Live demo

Open the project on GitHub Pages (after deploying `main`):

**https://hegschan.github.io/Asgn5/**

Use `index.html` for the site root, or `asgn5.html` if your course requires that filename.

## Controls

| Input | Action |
|--------|--------|
| Left-drag | Orbit camera |
| Scroll | Zoom in / out |
| Right-drag | Pan |

## Run locally

ES modules require a local server (do not open the HTML file directly with `file://`).

```bash
cd Asgn5
python3 -m http.server 8765
```

Then visit:

- http://localhost:8765/
- http://localhost:8765/asgn5.html

## Scene highlights

- **Floating house** with 128 pastel balloons
- **Paradise Falls** : textured cliff, flowing waterfall
- **Spirit of Adventure blimp** : custom OBJ+MTL model in the left sky (visible on load)
- **Procedural bird** with articulated wings
- **Yard figures** : old man, boy, golden retriever
- **Doberman patrol** : villain-dog style figure circling the red berries with animated legs
- **140 trees** and **155 rocks** scattered across the map
- **42 cloud** clusters

```
    ├── generate_waterfall_texture.py

```
