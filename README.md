# CSE160 Assignment 5 — *Up* Adventure World

A Three.js 3D scene inspired by Pixar’s [*Up*](https://www.pixar.com/up): Carl’s balloon-lifted house, Paradise Falls, a Spirit of Adventure–style blimp, animated wildlife, and a large explorable environment.

## Live demo

Open the project on GitHub Pages (after deploying `main`):

**https://hegschan.github.io/Asgn5/**

Use `index.html` for the site root, or `asgn5.html` if your course requires that filename.

## Controls

| Input | Action |
|--------|--------|
| Left-drag | Orbit camera |
| Scroll | Zoom in / out |
| Right-drag | Pan |

## Run locally

ES modules require a local server (do not open the HTML file directly with `file://`).

```bash
cd Asgn5
python3 -m http.server 8765
```

Then visit:

- http://localhost:8765/
- http://localhost:8765/asgn5.html

## Assignment requirements

| Requirement | Implementation |
|-------------|----------------|
| Three.js scene (beyond basic demos) | Full animated world in `main.js` |
| ≥3 primary shape types, animation, directional light, perspective camera | Boxes, spheres, cylinders, cones, planes, etc.; house/balloons/bird/waterfall animated; `DirectionalLight`; `PerspectiveCamera` |
| Textured primary shape | House/porch (`hardwood.jpg`), cliff/rocks (`cliff_rock.jpg`), waterfall (`waterfall.jpg`) |
| Custom textured 3D model | `SpiritBlimp.obj` + `SpiritBlimp.mtl` (OBJ+MTL); `Duck.glb` (GLB) |
| Camera controls | `OrbitControls` |
| ≥3 light types | `AmbientLight`, `HemisphereLight`, `DirectionalLight`, `PointLight`, `SpotLight` |
| Skybox | Photographic sky (`sky_equirect.jpg`) + six cubemap faces in `textures/skybox/` |
| ≥20 primary shapes | Hundreds of meshes (trees, rocks, balloons, clouds, figures, etc.) |

## Scene highlights

- **Floating house** with 128 pastel balloons
- **Paradise Falls** — textured cliff, flowing waterfall
- **Spirit of Adventure blimp** — custom OBJ+MTL model in the left sky (visible on load)
- **Procedural bird** with articulated wings
- **Yard figures** — old man, boy, golden retriever
- **95 trees** and **110 rocks** scattered across the map
- **42 cloud** clusters

## Project structure

```
Asgn5/
├── asgn5.html          # Course entry page (import map + main.js)
├── index.html          # GitHub Pages entry (same scene)
├── main.js             # Scene, lights, loaders, animation
├── models/
│   ├── SpiritBlimp.obj # Custom blimp mesh
│   ├── SpiritBlimp.mtl
│   ├── Duck.glb        # Khronos sample model
│   └── README.md
├── textures/
│   ├── skybox/         # Cubemap + equirectangular sky
│   ├── hardwood.jpg
│   ├── cliff_rock.jpg
│   ├── waterfall.jpg
│   └── blimp_*.jpg
└── scripts/            # Regenerate assets (optional)
    ├── generate_blimp_model.py
    ├── generate_waterfall_texture.py
    └── import_realistic_skybox.py
```

## Tech stack

- [Three.js](https://threejs.org/) **v0.172.0** (CDN import map in HTML)
- `OrbitControls`, `GLTFLoader`, `OBJLoader`, `MTLLoader`

## Regenerating assets (optional)

Requires Python 3 with `numpy`, `Pillow`, and (for skybox) `py360convert`, `scipy`:

```bash
pip3 install numpy Pillow py360convert scipy

python3 scripts/generate_blimp_model.py
python3 scripts/generate_waterfall_texture.py
python3 scripts/import_realistic_skybox.py
```
