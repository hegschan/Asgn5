# Background music

## Active track

| File | Title | Notes |
|------|--------|--------|
| `married-life-self.m4a` | *Married Life* (SelfMusic) | Original recording by the project author — used as scene background music |

Configured in `music.js` as the sole `<source>`.

## Replacing the track

1. Add or overwrite a file in this folder (e.g. `married-life-self.m4a`).
2. Update `MUSIC_SOURCES` in `music.js` with the filename and MIME type (`audio/mp4` for `.m4a`, `audio/mpeg` for `.mp3`).

## Copyright note

Do not commit commercial soundtrack rips (e.g. the official Disney/Pixar recording) unless you have redistribution rights. The bundled SelfMusic file is the author’s own audio for this assignment.
